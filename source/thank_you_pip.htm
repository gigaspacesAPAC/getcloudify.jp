Thank you for downloading the Cloudify CLI script.

Below are the first tips you need for getting started.


  Install Cloudify on Other OSes

Prerequisites

To install using the script, you must have Python 2.7.x preinstalled.

{%note title=Prerequisites Installation%}By default, this script will not install any prerequisites (like pip, or any required compilers). You can supply it with the --force flag which will install all prerequisites without prompting you for anything other than a sudoer password (if required). {%endnote%}


    Click here to download the Get-Cloudify.py script (if the download doesn't automatically begin within a few seconds).
  Open a terminal, then, run  sudo python get-cloudify.py on Linux or OS X and python get-cloudify.py on Windows.
  After the installation is complete, try running  cfy -h in your terminal.

For Linux

To install via rpm packages, you must have Python2.7.x and pip 1.5+ installed and Python2.7.x must be executable as python from the path.

Continue reading the docs to get fully up and running.
